Comparación de versiones

Demo vs. Versión Completa

Estas son todas las limitaciones de la versión de demostración que se eliminan al instalar la versión completa.

Función ⚠ Versión Demo ✓ Versión Completa
Vista satelital Limitada — muchas regiones sin detalle Zoom libre en todas las regiones sin restricción
Exportar DXF Pierde las coordenadas originales Exporta con coordenadas UTM originales (geolocalización real)
Copiar calles ✗ No disponible ✓ Ctrl+C / Ctrl+V
Cerrar polígono Solo con el botón "Cerrar" Con botón Cerrar o presionando Enter
Importar archivos Solo DXF (AutoCAD) DXF + KML + KMZ (Google Earth / GPS)
Plano de impresión ✗ No disponible ✓ Plano A0 en PDF + imagen PNG para compartir
Guías DXF visibles ✗ No disponible ✓ Líneas y polilíneas de referencia visibles en canvas
Dimensiones de lotes ✗ No se muestran ✓ Dinámicas — se actualizan con el zoom
Al cerrar el programa Se cierra sin avisar (se pierde el trabajo) Pregunta si deseas guardar antes de salir
Barra de escala ✗ No disponible ✓ Barra de escala gráfica dinámica según el zoom
